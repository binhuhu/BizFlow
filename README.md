🌊 BizFlow: The AI-Native Business OS
Stop chatting, start running.
BizFlow 是一个基于 PNSA (Parametric-Nodal-Supervisor-Auditor) 架构的开源 Agent 操作系统。它将复杂的业务 SOP 转化为受控的参数化节点流，确保 AI 在执行任务时具备工业级的确定性与安全性。
🚀 核心特性
确定性逻辑：基于 BPNL 协议，将 AI 约束在预定义的业务节点中。
三权分立：Executor (干活), Supervisor (检查), Monitor (熔断) 角色分离。
具身执行：原生集成 OpenClaw (browser-use)，让 AI 具备物理网页操作能力。
自进化：自动记录 Case 账本，通过人工干预闭环实现逻辑进化。
